Gamification System

Running a business feels like playing a strategy game.

Every agent earns XP, levels up, and glows brighter as your store performs. Watch your crew evolve from rookies to Legend-tier automators.

XP & Leveling

L1–50 Progression

Foundation (L1–10) → Operative (L11–20) → Commander (L21–30) → Elite (L31–40) → Legend (L41–50). Higher levels unlock self-initiated tasks, cross-agent collaboration, and autonomous revenue optimization.

Glow States

Visual Productivity Signals

Each agent has a per-character glow color that scales with XP tier — dim at L1, steady pulse while processing, blazing corona at L30+. Red alert pulse fires when an agent is idle or a listing underperforms.

Dashboard

Command Bridge Live View

Real-time view of all 7 agents: XP bar, active task, streak counter, glow state. Shop metrics panel shows total listings, daily views, conversion rate, and revenue. Crew leaderboard shows top contributor each week.

Streak Multiplier

7-day streak: 1.5× XP  ·  30-day streak: 2× XP

Legend Tier Unlock

L41–50: Fully autonomous revenue optimization — zero human input needed
